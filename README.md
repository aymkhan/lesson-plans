# Aymen Khan
# Dec 2025

This program allows teachers to easily retrieve lesson plans. 
The lesson plans are originally written by the teachers in Google docs (.docx). 
The .docx files are read by this program and the teachers can perform READ and WRITE operations on the existing lessons. 
The lessons are covered for the following courses:
- ASL1
- ASL2
- ASL3

The lesson plans are categorized by Lessons and Activities.
